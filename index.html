Que es un elemento HTML

Un elemento esta definido por una etiqueda de inicio seguido de algun contenido y una etiqueta de finalizacion
  1. <Nombre de la etiqueta>:etiqueta de inicio
  2. <Contenido>:entre la etiqueta de inicio y de finalizacion
  3. </ Nombre Etiqueta>:etiqueta de finalizacion

Etiqueta < Map >

La etiqueta < Map > define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distintas areas detectables por el navegador, logrando identificar de manera independiente cada una de las areas

La etiqueta < map > define las areas de la imagen mediante la etiqueta < area > con los atributos HTML "shape" que define la forma del area señalada en la imagen, "coord" que define las coordenadas de cada uo de los puntos de la shpae, "href"
que puede definir hipervinculo para cada una de las areas

ejemplo:
<img src="workflow.jpg" usemap="#workmap" >
<map name="workmap">
<area shape="rect " coord = "34,44,270,350" href="pagina1.com">
<area shape="circle" coords="200, 172, 333, 250" href="pagina2.com"> </map>
